# Progetto Backend
## Istruzioni

1. **Installare il paccheto express **:
    ```bash
      npm i express
    ```

2. **Installare il paccheto nodemon **:
    ```bash
      npm i nodemon --save-dev
    ```
3.**Digitare il segeunte comando per fare lo start del server**:
    ```bash

      npm start
    ```
